Name Mappings

Web browser errors are reported in the Javascipt console by URL. A developer needs to fix the error in a file. Therefore every URL needs to be mapped to a file name. Javascript functions in the uri_map.js file perform this mapping. First match wins. In this early version of fireclipse, the developer needs to write and update these functions. Click "Edit Mapping Functions" to begin.
The URI to Filename mapping functions are read when this page is loaded. To reload them when you make changes to the mapping, select the Reload URI to Filename map button.
